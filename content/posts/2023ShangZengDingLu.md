## 《熵增定律》读书笔记 {#2023ShangZengDingLu}


### 第一章前言&amp;第二章熵增定律 {#第一章前言-and-第二章熵增定律}

-   **熵增定律指出，在孤立系统重，如果没有外力做功，那么整个系统就会退化为死气沉沉、毫无生气的一团物质。哪怕是太阳、星系甚至整个宇宙都不例外。**
    -   比如一间房间杂乱无章，此时，房间里充满了熵。如果要收拾房间，使它变得井然有序，那么你就需要耗费能量做功。 **因此，要做到熵减，把无序变成有序就需要消耗能量，只有不断有系统外的“外力做功”，不断地降维输入能量，那么结局才会大为不同。**
    -   个人如果能理解熵增定律，如果能合理使用一些与熵增定律相关的定律和法则，那么就有机会从原有的人生系统里跳出来，获取系统之外的能量。克服熵增做功，实现个人的成长飞跃，是“有策略地变成更好的自己”。
-   **如果只是想着在原有系统里去寻求突破往往是徒劳无功的，经过屡次尝试之后，我才意识到，更有效的办法是从系统外获取能量。这种领悟让我醍醐灌顶，我开始疯狂地通过阅读这种性价比最高的方式来“从外部世界摄入新能量”，不断强化自己。**
-   **改变的本质，其实是创造新经验，用新经验代替旧经验。创造新经验需要通过新的方法，获得新的反馈、新的强化，并切身体验它。**
-   什么是“熵增定律”
    -   熵增定律是对热力学第二定律的一种表述方式，简单地说，熵增定律是指一个孤立系统，即非活的系统如果被独立开来，又或者把它方知道一个均匀的环境重去，环境中的所有运动都会因为摩擦力的存在而在极为有限的时间里停下来，水从高处流到地处，高温也会变为低温并达到平衡。最终，整个系统退化为死气沉沉、毫无生气的一团物质。
    -   **熵增定律有效需要同时满足两个条件，缺一不可：一个是没有外力做功，另一个是孤立系统。因此，如果有外力做功和系统开放，就能对抗熵增，实现负熵。**
-   Rudolf Julius Emanuel Clausius在1850年提出： **出了能量守恒定律之外，另外需要补充一条定律，即：没有某动力的消耗或其他变化，不可能使热从低温转移到高温**
-   关于熵增定律的推论————假设宇宙是一个孤立系统，因此宇宙中的熵会趋于极大，并最终达到热平衡状态。也就是说，宇宙中的每个地方的温度都相等。宇宙最终归于熵死。————但根据现有证据，宇宙并不是一个孤立系统，并且物理定律都是有适用范围的，在有引力作用下的热力学和无引力作用下的热力学得出的结论完全不同。目前根据证据，宇宙膨胀的根本原因是引力作用，因此宇宙是处于不稳定状态的。
-   **对抗熵增的两条路径**
    -   建立耗散结构
        -   耗散结构是由化学家普利高津提出的一个理论，是一个远离平衡态的开放系统。如果一个人处于持续不平衡或者不均匀的状态中，他就不会陷入熵死的状态。耗散结构是一种型，它的本质依旧是外力持续做功
        -   一个人如果每天都跑步去锻炼身体，这就是一种典型的耗散结构。因为身体的能量通过跑步耗散。
        -   同样地，一个人如果不停地突破自己的舒适区，通过学习、输出和使用他所学到的知识和技能也是一种耗散结构。 **因为在输出和使用知识或技能的时候，这些学识、认知、技能会逐步内化为个人的行为和习惯，帮助他跃迁到更高的认知层级，拥有更美好的人生。**
    -   避免路径依赖
        -   建立耗散结构，个人还应当避免路径依赖，这是一个不断自我强化的过程，不能让人轻易走出来。
        -   **路径依赖的额本质是封闭，在原有的系统里不停的循环。在找寻新路径的过程中，必然需要突破原有的系统，到一个更大、更开放的系统中去探路。**
            -   比如20世纪80年代，在中国出租车司机是一个很赚钱的职业。但30多年过去了，会开车的人越来越多，当年20多岁的司机如果陷入路径依赖中，还在以开出租车谋生，他的情况就会每况愈下。
-   **真实的世界是非连续性的**
-   最好的状态————“精神负熵”
    -   精神熵：因资讯对意识的目标构成威胁而造成的失序状态。精神熵会导致自我解体，令人的效率大打折扣，让人无法集中注意力实现任何目标。
    -   资讯是否会对人产生精神熵和自我对这些素材如何理解有关。例如股市下跌的消息会让银行家担忧，但对异议分子来说却是振奋人心的好消息。
        -   **虽然一条新资讯有可能会让人不得不付出心力去应付，造成意识上的失序，但同时也可能会强化人们的目标，激发出个人更多的精神能量**
    -   **专注是产生精神负熵的关键因素。在精神负熵（心流）状态下反而能减轻脑力负担**
    -   **想要获得精神负熵，个人专注的这件事情还必须要有一定的挑战性，而且挑战难度要适中，就像下棋时棋逢对手**
-   **华为通过“研发做功”累积势能，再将势能转化为动能横扫市场。** 这个过程就相当于将一块巨石推到万仞之巅，然后轻轻一推。用强劲的动能在竞争中获得绝对优势。
    -   **对人性的好逸恶劳，任正非用学习做功；面对企业的落后，华为用研发做功**


### 第三章：成长型思维 {#第三章-成长型思维}

-   最容易成功的方法论————不断挑战才能成长
    -   一个实验：安排两组孩子做同一个拼图游戏，当他们做出来后，夸其中一组的孩子聪明，夸另一组的孩子努力。结果表明，被夸努力的孩子在面对难题时表现的比被夸聪明的孩子更好。
        -   **无论男女，只要被夸聪明，他们绝大多数都会陷入固定型思维，为了保持聪明的形象，他们变得保守，不愿意接受挑战，甚至不愿意学习新知识；而被夸赞努力的孩子，他们有很大概率成为成长型思维的拥有者，他们积极努力，愿意接受挑战，乐于用努力去解决问题，争取好的结果。成长型思维的拥有者总是具有更强的复原能力，更高难度的挑战能让成长型思维者兴奋，因为有学习和成长的机会，而且正是因为挑战难度大，所以即使挑战失败了，也并不意味着自己是笨蛋，反而意味着自己有更广阔的的成长空间，自己的潜能也更有可能被激发出来。因为这让他们看到了自己与挑战目标之间的差距，这种差距的张力便成了成长型思维者训练和努力的内在动力。**
-   固定型思维的人会觉得“江山易改、本性难移”，人的特质是天生的，是后天无法改变的。他们害怕被否定被嘲笑，所以面对挑战他们会把自己封闭起来，拒绝挑战。 **固定型思维者长期处于一个封闭系统，很少有外力做功。**
    -   在面对选择时，她们往往偏向于挑选自己擅长的事情，遇到困难也会过早放弃。他们因为害怕失败，所以忽略了努力会有好结果的可能，从而限制了自己的成长。
-   **成长型思维不意味着仅仅只需要努力，成长型思维是针对包括努力以及后续的策略、反思、迭代，以及再次行动这整个过程————真正的成长型思维者一定懂得听取、学习不同的策略，然后用不同的策略在一个难题上反复推敲，找到更好的解决办法，用有效、高效的方式去解决各类卡点**
    -   比如，有些职场人不善言辞，觉得领导不喜欢自己，怎么办？那就可以尝试通过写一封邮件，总结自己每个月所做的事情、本月的反思以及下个月的计划并发给领导，尽量让自己的工作被领导看到
-   **成长型思维不意味着只盯着目标，目标不是结果，达成目标还需要设法找到路径和资源**
    -   比如每一个单身青年如果希望找到伴侣，那么平时就要多参加有许多异性在场的社交活动。


#### 构建成长型思维 {#构建成长型思维}

-   阻碍成长的四座监狱：成功的监狱、过去的监狱、表现的监狱和恐惧的监狱
    -   成功的监狱————害怕失败，心态因虚名而发生改变
    -   过去的监狱————陷入对过去的怀念，屈服于身体的衰老，放弃探索尝试新事物
    -   表现的监狱————也是为了虚名，只注重表面文章不做实事
    -   恐惧的监狱————害怕失败丢面子、完美主义、害怕与别人冲突导致拖延、踟蹰不前
        -   **拖延会导致失去稍纵即逝的机会，害怕与别人冲突并不能消灭冲突。想要获得成长型思维，必须要勇敢，敢于直面恐惧**
-   5方面培养成长型思维
    -   有清晰、远大的目标
    -   勇于行动
        -   **先行动再幻想，因为只有反馈永远出现在行动之后，只有勇于行动的人才能获得反馈，只有获得反馈才能调整策略重新行动**
    -   坚持
        -   **坚持背后的底层动机是对自己的信心**
    -   积极乐观
        -   乐观心态促进行动，行动产生反馈让心态更积极，形成正向循环
    -   承担责任
        -   欲戴皇冠，必承其重。逃避自己应当承担的责任是无法成长的，在困难压力中磨砺自己才能获得成长


#### 做一个终身学习者 {#做一个终身学习者}

-   **成长型思维的人具有很强的复原力。如同邓亚萍，多次被教练认为先天不足，但依然没有放弃，不停地提升自己。**
-   **克服压力的方法————回到当下，进入过往不恋、未来不迎、当下不杂的状态**
-   **任何事情，你都是从不会到会，从会再逐渐地去感悟和提炼这个成功的规律————邓亚萍**
-   医生暗示艾瑞克森不可能站起来，但是艾瑞克森否定了医生的暗示，在绝大多数全身瘫痪的人都无所建树的情况下，靠自己探索出了催眠技术，并靠自己的努力站了起来


### 第四章：损失厌恶原理 {#第四章-损失厌恶原理}

-   **人需要一些心理学原理， 来与惰性抵抗，辅助增强做事的动机。损失厌恶原理是其中之一**

-   **损失厌恶原理：人们在面对相同数量的损失和收益时，损失会让人更难以接受。相同数量的负面效用在心理层面上的作用是正面效用的2-2.5倍**
    -   对于多数人来说，损失100元的痛苦远远大于获得120元的快乐——————当奖励达到200-250元时，才能抵消这种痛苦
    -   **损失厌恶原理的本质是一种认知偏差**
        -   比如人们很容易被“错过等一年”的广告套路，又比如投资时，由于厌恶损失，人们会不愿意抛售下跌的股票，却更偏向于将上涨的股票做获利了结。
    -   因害怕损失而改变
        -   训练人的专注力的方法：购买一件足够昂贵的学习工具————由于损失厌恶原理，人们会觉得“少用一会儿是种浪费”———— **买东西不能图便宜，图便宜会大大浪费一个人的时间** ————“因为便宜的东西本来就不是依靠专注力制造出来的，只有买来昂贵的物品才能让你感知到某种专注力，才能对一个人本身的专注力形成一种训练”
            -   **“钱都花了，不能白花啊是吧”**
        -   通过损失厌恶机制来早起
            -   第一步，准备2个闹钟或2部手机，如果期望自己6点起床，那么就把其中的一个闹钟设置为6点，并把它放在自己的卧室里；第二步，将另一只闹钟设置为6点03分，并把它放在客厅里；第三步，设置为6点的闹钟闹铃的音量可以适当地小一些，只要能把自己从睡梦中唤醒就可以了，而设置为6点03分的闹钟则要把闹铃的音量调至最大，大到足以把全家人都能闹醒。
            -   如此一来，第二天早上，等第一只闹钟把你唤醒之后，为了不让3分钟后会产生巨大闹铃声的闹钟触发（规避可能被家人骂的精神损失），你就有足够的动力起床，并去关闭那只即将响彻全家的闹钟了。而这么起来走一圈，很大程度上睡意就会消失，早起的目标也就达成了。
            -   我自己的想法是，可以通过在6:03来做早饭，比如设定蒸蛋机在6:03蒸鸡蛋，来倒逼自己早起
-   星星法则，做出持久改变
    -   星星法则：马丁迈德斯提出的结构化坚持模型，包括Start、Traction（惩罚）、Adherence（坚持）、Refinement（修正），即STAR四部分。
    -   例如你可以把1000元给朋友，并告诉他如果自己再碰零食这1000元你就不用还了。
-   如何避免损失厌恶心理对自身的负影响————多做慢思考
    -   在快思考的时候，人会偏感性，它是人的本能反应，也是平时经验积累的结果，系统的运作是快速且无意识的，决策往往受到包括损失厌恶心理在内的很多感性因素的影响。慢思考则是理性的，通过学习、计算、逻辑推理等需要花费大量时间和脑力深入思考，也就是常说的深思熟虑。
    -   例如，一个规避损失厌恶原理的好方法就是从做股票交易转行到做行业基金投资，因为行业基金投资需要24小时以上才能完成，所以可以有充分的时间调研和思考，更可能避免损失厌恶心理情绪的影响。
-   通过“损失”激发行动
    -   当网约车司机想要关闭APP下班时，App会弹出提示“你还差87元就能赚到500元了，你难道真的想现在就下线吗？”————————这条信息会让司机觉得我本该赚到500元，从而通过损失厌恶心理来促使司机继续工作。
    -   在线学习平台通过打卡学习满30天全额返还学费的规则，来利用学生的损失厌恶心理调动学生学习的积极性
    -   另一些在线学习平台，在学生免费试学期间，会对学生作对题目等给出金币奖励。当免费试学结束后，会告诉学生们只要报名学习，免费试学期间获得的金币可以兑换奖品。这也是利用损失厌恶原理来调动学习积极性。
    -   **损失厌恶原理经常倍用在市场营销领域，主要有以下4种营销方法**
        -   限时秒杀
        -   席位有限
        -   满减包邮
        -   限时涨价
-   在股市中，损失厌恶原理经常会让你“买股深套”或者“卖了就涨”
    -   对于那些长期增幅达到5倍、10倍的绩优股，在长期涨幅过程中，也会存在30%-50%的巨幅震荡。出于对 **获利的损失厌恶** ，会 把手中的股票卖出，造成“卖了就涨”的现象；
    -   对于那些一直下跌的股票，由于股民出于对卖出股票造成的 **实际发生的亏损** 的厌恶，会舍不得卖出，一直期盼牛市到来


#### 我的感悟 {#我的感悟}

**这对我的启示还有一层————我们一定要做一个长期主义者————如果一个股票长期是看好的，那么即使中间发生短暂的下跌，也不要卖出股票，否则会导致“卖了就涨”————如果一个股票长期是看跌的，那么即使中间发生涨幅，也不要心存幻想，赶紧卖出才是最好的选择——————对于事情来说也是一样，如果一件事情远期看是获利的，那么你就去做；远期看是有害的，不要被眼前的蝇头小利而蒙蔽，及时放弃才是好的。**


### 第五章：从众效应 {#第五章-从众效应}

-   从众效应和损失原理不一样，它产生的不是动机，而是一种触发
-   线段实验
    -   这项实验以大学生为受试对象，实验人员把大学生每7人分为一组，围坐在一起，然后向每组学生出示A、B两张卡片，其中A卡片上有一条线段，而B卡片上则是长短不一的三条线段。实验人员要求受试者逐一判断B卡片的三条线段中，哪一条和A卡片的线段长度最接近。在阿希教授的实验中，他把一次实验分为18场，前6场按照真实情况进行，哪根长度接近就直接说出正确的那根；但从第7场开始画风急转，前6名同学是被安排好的托儿，他们开始故意带节奏，一个个报出错误答案来误导实验对象。现实中，明眼人一看就能判断出，长度最相近的应该是线段C，但实验人员提前安排好的托儿却故意说错，而且还一致地给出一个错误的答案。与此同时，实验人员开始观察这第7位真实的受试者是否会发生预想中的从众行为。结果显示，被试者的反应可以被总结为如下三类。

        -   第一类：这类受试者从头到尾始终如一地坚持自己的判断，这类人占24%。第二类：在12次实验同学故意说错的场次中，他们至少会发生一次从众行为，这类被试者占71%。第三类：自始至终始终从众的受试者，占5%。 **从总体结果来看，平均有三分之一的人会有从众行为的发生。**

        用信任和期望激发善意，获得积极反馈，激活你在这门课程上的学习动力。
    3.  小心罗森塔尔的负向效应
        就是小心负面的心理暗示对自己的影响。例如有些人总觉的自己“不是读书的料”，那么就算有时间学习他们也觉得学习没用从而把时间浪费到其他事情上；又或者情侣吵架时，其中的一方可能会说“你看你，总是把自己活成一座孤岛”“你总是不在乎我的感受“—————— **被指责的一方可能在这些语言暗示下形成负面的自证预言，而且可能会激发出恶意，所以少使用这些负面指责语言，防止自己的境遇变得糟糕却不知道是负面的自证预言的原因。**
        -   破解负向罗森塔尔效应：事实＋感受＋请求
            -   场景：你的伴侣有一天不知道为什么，突然拿你和同事的伴侣做比较。一开始你忍住了这份不悦，没有表达出来，但接下来的谈话内容让你觉得越来越不对味，你也越来越压制不住自己肚子里的火，你的感觉就好像过去父母在讨论“别人家的孩子”一样。你原本也打算用对方做的不好的地方予以反击。但你突然意识到，这可能是负向罗森塔尔效应在作祟（ **意思是说对方指责你的这种言语暗示会让你觉得自己就是被指责的那样的人，这就是负面的心理暗示产生的危害** ），那自己有没有办法破解它呢？于是，你说：“亲爱的，你刚才拿你同事的伴侣和我做比较，让我感觉小时候我的爸妈在和我讨论‘别人家的孩子’，现在我心里很难过，请你别再拿别人和我做比较了。”
            -   **当指责别人的一方听到这样的表达后，会意识到自己刚才“作比较”的行为有问题，然后停止这样的行动————————这种方法出自马歇尔卢森堡的著作《非暴力沟通》，即将双方语言背后的思想摊在桌面上，从而让人能意识到并能让人做出行为改变的技术。————整个话术分成1.事实2.感受3.请求三部分**
                -   事实，即把之前发生的事情说出来，在以上场景中即为“你拿同事的伴侣和我做比较”
                -   感受，即事实发生后自身的感受变化，要用平静的语气表达自己最真实的感受：“这样的比较让我很不舒服”
                -   请求，因为自己的感受很糟糕，所以请求对方停止这样做：“请你别再拿别人跟我作比较了”
        -   改变你的口头禅（跟武志红“你常说的一句话正在悄悄改变你的命运”异曲同工）
            **即在自我调整的过程中，提升觉知，让自己的“潜意识”被显现，从而防止自己陷入负向罗森塔尔效应。口头禅会成为潜意识的语言外化，并通过不停吐露来不断强化，继而变成现实**
        -   乔哈里视窗：观测潜意识的工具

            | 区域 | 含义        | 作用                                        |
            |----|-----------|-------------------------------------------|
            | 开放区 | 自己和别人都知道的信息 | 在实际工作中的人际交往中，共同的开放区越多，沟通起来也就越便利，越不易产生误会。 |
            | 盲目区 | 自己不知道别人知道 | 地位和权势越高的人，越难听到关于自己的真话，这时因为周围都是奉承者，沟通单向闭塞。 |
            |     |             | 因此没有博大的胸怀容纳一些敢于讲真话的朋友、同事、下属，盲目区就会月越来越大。 |
            | 隐秘区 | 自己知道、别人不知道 | 一个真诚的人也需要隐藏区，完全没有隐藏区的人是心智不成熟的。 |
            |     |             | 但在有效沟通中，适度地打开隐藏区，是增加沟通成功率的一条捷径。 |
            | 未知区 | 自己和别人都不知道 | 也许通过某些偶然或必然的机会，得到了别人较为深入的了解，自己对自我的认识也不断的深入。 |

            -   **我们的目的是通过乔哈里视窗把自己的盲目区看清楚————依赖从别人那里获得的反馈来更好的了解自己**

            | 区域 | 技巧                                                           |
            |----|--------------------------------------------------------------|
            | 开放区 | 要想使你的公开区变大，就要多说，多询问，询问别人对你的意见和反馈。多说、多问不仅是一种沟通技巧，同时也能赢得别人的信任。 |
            | 盲目区 | 盲目区是不足之处别人看得见自己却不知道，多说、多问不仅是一种沟通技巧，同时也能赢得别人的信任。要多说而且要多问，避免盲区过大 |
            | 隐秘区 | 内心封闭的人别人都无法信任，还是要多问多说，主动告诉别人一些自己的信息 |
            | 未知区 | 封闭会失去很多机会，能够胜任的工作可能就从身边悄悄溜走了，要主动向别人介绍自己，主动的告诉别人自己能够做什么 |
